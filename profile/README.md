## 👋 你好
Open UUP dump 是一个致力于恢复 UUP dump、TechBench dump 等项目透明度的开源组织。

我们认为，真正的安全不应以牺牲透明度为代价。  
以 “安全” 之名闭源，往往只是制造了不可审查的黑盒。  
这使得用户无法审查源代码，剥夺了用户验证安全性的权利。

我们坚信，只有保持开放、协作、共享，才能延续一个项目的生命力。  
因此，我们收集了这些项目已知的最后开源代码，对其进行适当的维护后，重新开源。

## 👋 Hello
Open UUP dump is an open-source organization dedicated to restoring transparency to projects such as UUP dump and TechBench dump.

We believe that true security should never come at the expense of transparency. 
Closing source code in the name of "security" often does nothing more than create an unauditable black box.  
This prevents users from inspecting the source code and deprives them of their right to verify security for themselves.

We firmly believe that only through openness, collaboration, and sharing can a project’s vitality be sustained.  
Therefore, we have collected the last known open-source code for these projects and, after proper maintenance, have made them open-source once again.
